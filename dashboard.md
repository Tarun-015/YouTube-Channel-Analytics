## 📁 Dataset Summary
The dataset contains basic metrics for each video:
- 🎬 **Video Title**
- 📅 **Upload Date**
- 👁️ **Views**
- 👍 **Likes**
- 💬 **Comments**
- 📈 **Dislikes**
- 👥 **Subscribers**

> *(Dataset is based on either self-recorded channel data or synthetic data for visualization purposes)*
